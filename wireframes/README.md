# FoodIntel AI - Figma Wireframe Pack

This folder contains Figma-ready SVG wireframes for the FoodIntel AI submission.

Frames included:
1. Login by sales category
2. Command Center
3. Prospect Brief
4. AI Copilot with RAG + web/proxy-search answer structure
5. AI Workflow
6. Follow-up Studio

How to use in Figma:
1. Open Figma.
2. Create a new design file.
3. Drag all SVG files into the canvas.
4. Arrange them left to right as a prototype flow.
5. Optional: add prototype links between frames in this order:
   Login -> Command Center -> Prospect Brief -> AI Copilot -> AI Workflow -> Follow-up Studio.

Submission wording:
The wireframes were created to validate the key FoodIntel AI workflow before the working React MVP: category login, active prospect context, meeting prioritization, prospect briefing, RAG-based AI Copilot reasoning, agentic workflow visibility and post-meeting follow-up.
