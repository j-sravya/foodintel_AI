# PostHog Event Plan

Track these product events to prove user value and improve the MVP.

| Event | When It Fires | Why It Matters |
|---|---|---|
| `category_login_completed` | User logs into a sales category | Shows activation |
| `command_center_viewed` | User opens dashboard | Measures daily usage |
| `prospect_selected` | User changes active prospect | Tracks workflow intent |
| `brief_generated` | Prospect brief is opened/generated | Core value metric |
| `pitch_downloaded` | Word pitch is downloaded | Strong buying-intent event |
| `followup_generated` | Follow-up message is created | Measures post-meeting automation |
| `copilot_question_asked` | User asks AI Copilot | Measures AI engagement |
| `source_evidence_opened` | User views trust/evidence data | Measures explainability usage |
| `subscription_checkout_started` | User starts Stripe checkout | Revenue funnel event |

## North Star Metric

```text
Meeting-ready briefs generated per active sales rep per week
```
