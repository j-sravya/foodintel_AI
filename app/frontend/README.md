# React Frontend

This folder is the target production frontend for FoodIntel AI.

Recommended stack:

```text
React
Vite
TypeScript
Tailwind CSS
Supabase JS client
PostHog JS client
Stripe checkout redirect
```

## Main Screens

- Category Login
- Command Center
- Prospect Brief
- Opportunity Radar
- CRM Timeline
- Follow-up Studio
- AI Copilot
- AI Workflow

## Global Context

Every page should read the same active context:

```text
Seller company
Logged-in sales category
Active prospect
Active meeting
Product focus
Plan access
```
