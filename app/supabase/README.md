# Supabase Backend

Supabase should handle:

- Authentication
- Company users
- Seller categories
- Product catalog
- Prospects
- Meetings
- Source registry
- Raw signals
- Insight audit logs
- Product match scores
- Competitor mentions
- Account history
- Brief versions
- Follow-up actions
- Subscription status from Stripe

Use row-level security so a company can only see its own users, prospects, meetings and intelligence.
